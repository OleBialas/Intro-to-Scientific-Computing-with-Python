Different Approaches

Seaborn - Declarative

  • Describe what data you want to plot
  • "Plot variable 1 against variable 2 and add a color code for variable 3"
  • Complex graphs with little effort
  • Customization is limited

Matplotlib - Imperative

  • Describe how to plot the data
  • "Draw red circles at these coordinates and blue circles at these coordinates"
  • Fine grained control
  • Creating complex graphs is laborious
