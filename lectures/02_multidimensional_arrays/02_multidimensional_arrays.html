Working with Multidimensional Arrays in Numpy

Why Numpy?

  • Written in C, highly performant
  • Allows vectorized operations
  • Basis of the scientific ecosystem
  • Handles mutlidimensional data

Multidimensional Data

  • Scientific data often has multiple dimensions (time, sensors, repetitions, subjects etc.)
  • Each dimension is represented in a separate axis

Epoched EEG Data

  • Same stimulus is displayed repeatedly
  • Recordings are cropped to a fixed epochs
  • Represented as 3-D array (epochs, channels, times)
